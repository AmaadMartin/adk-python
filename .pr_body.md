**Problem**: Intermediate, unpersisted state keys (prefixed with `temp:`) were being included in session checkpoints during `_flush_deltas` and `_flush_output_and_deltas` steps in `_node_runner.py`.

**Solution**: Added a filter to exclude any state keys starting with `temp:` before they are appended to the `state_delta` in the `EventActions`. Cleared `state_delta` correctly after extracting the filtered delta to prevent memory leaks, and added corresponding unit tests in `test_node_runner_ctx.py`.
